'Nanna' - a Max UI for CLM Granular Syntheis

This software package allows users to control the grani CLM instrument by Fernando Lopez-Lezcarno using Max/MSP.

Envelopes for key grain parameters can be specified for up to 5 simultaneous grain streams using XY breakpoint graphs. This approach combines the high audio quality of CLM's offline processing with the intuitiveness offered by Max/MSPs UI objects.
